# Template Images

Đặt các ảnh template ở đây để dùng cho auto render (chụp từ CapCut UI):

- `home_ready.png` - Màn hình home CapCut
- `btn_export.png` - Nút Export trong editor
- `btn_start.png` - Nút Start trong popup export
- `popup_export_check.png` - Dấu hiệu export dialog đã mở
- `rendering_check.png` - Dấu hiệu đang render

## Cách tạo template

1. Mở CapCut
2. Dùng Snipping Tool chụp từng nút/UI element
3. Lưu dưới dạng PNG vào thư mục này
4. Cắt sát viền nút, không lấy quá nhiều background
