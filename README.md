# CapCut BVM

Phần mềm tự động hóa CapCut: tạo phụ đề bằng AI, batch export SRT, inject vào draft, auto render.

## Cài đặt

```bash
pip install -r requirements.txt
```

## Chạy

```bash
python main.py
```

## Đóng gói .exe

```bash
pip install pyinstaller
python build.py
```

## Tính năng

- 🎬 **Auto Render** - Tự động export video từ CapCut
- 📝 **Batch SRT** - Tạo file phụ đề .srt hàng loạt bằng AI (Whisper)
- ✨ **Manual FX** - Thư viện hiệu ứng
- 🌙 **Dark Theme** - Giao diện tối, nhẹ mắt
